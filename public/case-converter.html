All 8 Case Formats Explained

  • UPPERCASE — All letters capitalized. Great for headings and emphasis.
  • lowercase — All letters in lower case.
  • Title Case — First letter of each word capitalized. Standard for titles and headings.
  • Sentence case — Only the first word and words after punctuation are capitalized.
  • camelCase — No spaces, each word starts with a capital (except the first). Used in JavaScript variable names.
  • PascalCase — Like camelCase but the first word is also capitalized. Common in class names.
  • snake_case — Words separated by underscores, all lowercase. Popular in Python and databases.
  • kebab-case — Words separated by hyphens, all lowercase. Standard for URLs and CSS classes.
